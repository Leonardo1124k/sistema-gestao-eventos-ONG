# 🏠 Sistema de Gestão de Eventos integrado ao Site – ONG Moradia Digna

## 📌 Descrição do Projeto

Este projeto faz parte do **Projeto Integrador** do curso de **Análise e Desenvolvimento de Sistemas da Fatec Franca**, desenvolvido pelos alunos do Grupo G3 desde o primeiro semestre de 2025, com previsão de término no ultimo semestre de 2027.

No primeiro semestre o objetivo principal foi a criação de uma **landing page institucional para a ONG Moradia Digna**, com foco em facilitar o processo de doação, engajar novos voluntários e ampliar a visibilidade digital da organização.

No segundo e terceiro semestre o objetivo foi o desenvolvimento de um **Sistema de Gestão de Eventos**. O sistema visa facilitar o processo de **interesse de compra** de produtos por parte clientes e apoiadores da ONG durante eventos promovidos pela ONG (principalmente o Talharim Solidário). Alem de auxiliar voluntários e gestores no dia dos eventos, com **registro digital de retiradas e pagamentos das reservas**.

O sistema foi desenvolvido utilizando tecnologias web modernas e seguiu as melhores práticas de **Engenharia de Software**, passando por todas as etapas de levantamento de requisitos, modelagem de casos de uso, documentação, diagramação e desenvolvimento.

## 🛠️ Tecnologias Utilizadas

- HTML5
- CSS3
- Java
- Spring Boot
- MySQL
- Autenticação JWT 
- Spring Data JPA

## 📋 Funcionalidades Principais

- Permite planejar e executar múltiplos eventos, controlando o que será vendido neles.
- Focado em registrar os pedidos de forma segura.
- Garante que nenhuma reserva seja entregue sem a devida conciliação.
- Toda a operação do sistema é blindada para evitar acessos indevidos (segurança).
- Design responsivo (compatível com desktop e mobile).
